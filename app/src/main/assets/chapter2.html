Part III

Significant Figures

Significant figures in the measured value of a physical quantity tell the number of digits in which we have confidence.

Points to find significant figure

  • All non-zero digits are significant figure.
  • A zero becomes significant figure if it appears between two non-zero digits.
  • Leading zeros or the zeros placed to the left of number are never significant.
  • Trailing zeros or the zeros placed to right of the number are significant.
  • In exponential notation , the numerical portion gives the number of significant figures.

Rounding off rules

  1. If the digits to be dropped is less than 5 , then preceding digit is left unchanged.
  2. If the digits to be the dropped is more than 5, then the preceding digit is raised by one.
  3. If the digits to be dropped is 5 followed by digits other than zero, then the preceding digit is raised by one.
  4. If the digits to be dropped is 5 or 5 follwed by zeros , then preceding digit is left unchanged , if it is even.
  5. If digit to be dropped is 5 or 5 followed by zeros , then the preceding digit is raised by one , if it is odd.

Significant figure in calculation

1.The result of addition or subtraction in the number having different precisions should be reported to same number of decimal places as present in the number having least number of decimal places . 2. The answer to a multiplication or division is rounded off to the same number of significant figres as possesed by the least precise termused in the calculation.

Errors of Measurement

The measuring process is essentitally a process of comparision. Inspite of [our best efforts, the measured value of a quantity is always somewhat different from its actual value, or true value .The difference in the measured value and true value is called error of measurement.

1.Absolute error

Absolute error in the measurement of a physical quantity is the magnitude of the difference between the true value and the measured value of the quantity.

Let a physical quantity be measured n times. Let the measured values be $a_1,a_2,a_3.....a_n$. Thearithmetic meanof these values is ${a_m} = {a_1+a_2+a_3.....a_n}/n$.

By defination , absolute errors in the measured values of the quantity are
$Δa_1 = a_m - a_1$
$Δa_2 = a_m - a_2$
$Δa_3 = a_m - a_3$
.................
$Δa_n = a_m - a_n$ .
The absolutte error maybe positive and negative.

2.Mean absolute error

It is arithemetic mean of the magnitudes of absolute errors in the all measurements of the quantity.It is represented by $\ov{Δa}$.
$\ov{Δa}$ = ${|Δa_1|+|Δa_2|+|Δa_3|.....|Δa_n|}/n$
Hence the final result of the measurement maybe written as a = $a_m +↙{-}\ov{Δa}$.

3.Relative error or Fractional error

The relative error of measurement is defined as the ratio of mean absolute error to the mean value of quntity measured. $$Relative error = {Mean absolute error} / {Mean value} = {\ov{Δa}}/a_m$$

4.Percentage error

When the relative or fraction error is repesented in percentage , it is called Percentage error. Percentage error = ${\ov{Δa}}/{a_m}*100%$
